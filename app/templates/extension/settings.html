extensión/

Autofill de credenciales en el navegador, conectado a tu Dev Hub.

cómo conectar

1. Carga la carpeta extension/ del repo en chrome://extensions (modo desarrollador → "Cargar descomprimida").

2. Abre la extensión, pon la URL de tu Dev Hub y entra con tu email y contraseña.

3. Define tu PIN de desbloqueo (mínimo 4 dígitos). Listo: el ícono aparecerá en los formularios de login.

El login crea un token de acceso que aparece abajo. Si pierdes el dispositivo o sospechas algo, revócalo aquí y la extensión queda desconectada.

conectar claude code / mcp

Genera un token para que Claude Code lea y escriba en tu Dev Hub vía el servidor MCP. El token reemplaza tener que correr curl a mano.
